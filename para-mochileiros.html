Para Mochileiros

No Guia, fala-se que o mochileiro, viajante intergalático, deve estar preparado, afinal não se sabe em quais enrascadas você pode se meter. Mas afinal, o que todo o mochileiro deve levar consigo?

Toalha

Todo mochileiro deve ter uma toalha. No Guia, é descrito o quão amplo são os usos das toalhas, como: poder ser uma arma (se molhá-la, ela pode machucar), poder ser um capus contra o sol, poder quardar coisas fazendo uma trouxa, ser uma toalha normal, etc...

Ou seja, tenha uma toalha.
